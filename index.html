ThingML

ThingML is a modeling language for embedded and distributed systems. ThingML stands for "Thing" Modeling Language as a reference to the so called Internet of Things. It was developped by SINTEF (Oslo, Norway) and is now owned by Tellu IoT (Asker, Norway).


ThingML is an open-source project available under the Apache 2.0 License.


ThingML is developed as a domain-specific modeling language which includes concepts to describe both software components and communication protocols. The formalism used is a combination of architecture models, state machines and an imperative action language. The ThingML toolset includes text editors to create and edit ThingML models, a set of transformations to create diagrams from ThingML models and a set of code generators to compile ThingML to C (Linux and Arduino), Java and JavaScript (NodeJS and Browser).
